在Ubuntu中怎么安装JDK,并安装pycharm

必备技能

搭建在ubuntu上的pyton编辑器pycharm

步骤如下：

一.安装JDK：

    1. ubuntu使用的是openjdk，我门要首先找到适合的jdk版本。
    2. 在终端输入命令：apt-cache search openjdk
      如图

    3. 从搜索的列表里找到我们需要安装的jdk版本
      oracle-java9-installer - Oracle Java(TM) Development Kit (JDK) 9

    4. 执行安装命令：sudo apt-get install oracle-java9-installer

  1. 我们可以通过java－version命令来验证安装结果。

  2. 如果能看到上图的现实结果证明已经安装成功了。

二.安装pycharm：

    1. 官网下载地址：https://www.jetbrains.com/pycharm/download/
      这里的PyCharm有付费和免费版本，我们使用免费版本（Community Edition）就够用。

      当前的版本是pycharm-community-4.0.4，下载的文件是.tar.gz压缩，我们把下载的文件放到可知的盘符文件里，然后需要进行解压。

      解压完毕之后，在bin文件夹中pycharm.sh执行文件进行安装。安装的时候会提示”No JDK found. Please validate either PYCHARM_JDK, JDK_HOME or JAVA_HOME”的错误，看来暂时还不能安装，需要安装JDK等组件包环境才可以。

    2. 安装PyCharm工具
      目前环境已经配置完毕，然后我们再执行sh pycharm.sh文件安装。
