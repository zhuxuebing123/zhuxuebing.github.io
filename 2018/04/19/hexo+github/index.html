效果请看本人博客

  • 一：先下载需要的安装包：

  • 1.下载node.js包:

  • 2.按window+R键进入运行框输入cmd,进入命令提示符界面

    在命令提示符窗口输入：

    1 node -v

    2 nmp -v

  • 3.下载git安装文件：

  • 1.创建代码库

    1）登录之后点击右上角+号，选择New repository

    新建代码库

    2）在Repository name栏填写yourname.github.io（不能随便填），Description (optional)下填写一些简单的描述（不写也没有关系），如图所示：

    3）如果你要绑定域名，就必须先自己的域名，不绑定的可以跳过此步】

    注册域名的网站较多，像博主这种对流量要求不高的就在腾讯云上买了个便宜的域名，

    去域名管理中心点击”解析”,记录类型选择CNAME方式，

    记录值填yourname.githun.io

    点击保存大约10分左右就可以解析成功，达到绑定的目的了。

    如图我的绑定域名

    进入刚才创建的rerepository,点击Setting按钮，如图

    如图在github Pages页面

    在Custom domain填入自己的域名点击保存即可

      2)添加SSH Key

      检查 SSH keys的设置，首先我们需要检查你电脑上现有的ssh key：

      如果提示：No such file or directory 说明你是第一次使用git。

      生成新的SSH Key：

      Generating public/private rsa key pair.

      Enter file in which to save the key (/Users/your_user_directory/.ssh/id_rsa):<回车就好>

      注意1: 此处的邮箱地址，你可以输入自己的邮箱地址;注意2: 此处的「-C」的是大写的「C」

      然后系统会要你输入密码：

      Enter passphrase (empty for no passphrase):<输入加密串>

      Enter same passphrase again:<再次输入加密串>

      在回车中会提示你输入一个密码，这个密码会在你提交项目时使用，如果为空的话提交项目时则不用输入。这个设置是防止别人往你的项目里提交内容。
